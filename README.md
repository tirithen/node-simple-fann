# node-simple-fann
A Node.js wrapper for [FANN (Fast Artificial Neural Network Library)](http://leenissen.dk/fann/wp/) that uses MongoDB to store the models.

## Requirements
* libfann >= 2.2.0 - [Get source code here](http://leenissen.dk/fann/wp/download/)
